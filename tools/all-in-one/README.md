# All-in-One GIF Processing Tool

一个集成了 GIF 压缩、去重、重命名和批量处理功能的一站式工具，采用类似 MemeTray 主页的界面设计。

## 🚀 功能特性

- **📁 批量上传**: 支持拖拽文件夹、多文件选择、Ctrl+V 粘贴
- **🗜️ 智能压缩**: 自动将 GIF 压缩到 128×128 像素，支持 WebP 转 GIF
- **🔍 智能去重**: 基于像素内容检测重复 GIF，支持自动/手动删除
- **🏷️ 批量重命名**: 按 MemeTray 格式重命名 (XXXX_name.gif)
- **🖼️ 可视化预览**: 类似主页的网格展示，支持选择、预览、删除
- **📊 实时统计**: 显示压缩率、去重数量、重命名状态
- **⚙️ 灵活配置**: 可自定义流程步骤和参数

## 📋 工作流程

1. **上传文件** → 拖拽或选择 GIF 文件/文件夹
2. **压缩处理** → 自动压缩到标准尺寸
3. **智能去重** → 检测并标记重复文件
4. **批量重命名** → 按设定格式重命名
5. **结果展示** → 可视化展示处理结果
6. **下载导出** → 选择需要的文件打包下载

## 🎨 界面特色

- **Windows 11 风格**: 玻璃态效果、圆角设计、流畅动画
- **步骤指示器**: 清晰显示当前处理进度
- **网格预览**: 类似 MemeTray 主页的 GIF 展示
- **批量操作**: 支持全选、反选、删除选中
- **状态标记**: 显示压缩率、去重状态、重命名状态

## ⚡ 使用说明

1. 访问工具页面
2. 拖拽 GIF 文件到上传区域，或点击选择文件
3. 根据需要调整选项设置
4. 点击处理，等待工作流程完成
5. 在结果页面查看和选择文件
6. 下载处理后的文件

## 🔧 技术实现

- **前端**: 纯 HTML/CSS/JavaScript (ES6+)
- **图像处理**: Pyodide + Pillow (浏览器内 Python)
- **文件处理**: JSZip 打包下载
- **UI 框架**: Windows 11 风格玻璃态设计
- **图像分析**: Canvas API + 像素哈希算法

- 批量处理表情包 GIF
