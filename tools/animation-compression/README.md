# GIF 压缩工具

将 GIF 图片缩小到 32x32 像素，保持动画效果。

## 使用方法

直接打开 `index.html` 文件即可使用。

1. 第一次打开会加载 Python 环境（约 10-20 秒）
2. 点击上传区域或拖拽 GIF 文件
3. 等待处理完成
4. 点击下载按钮保存压缩后的 GIF

## 特点

- ✓ 完全在浏览器中运行，无需安装任何软件
- ✓ 使用 Python Pillow 库处理，兼容性最好
- ✓ 保持 GIF 动画效果和帧率
- ✓ 使用高质量缩放算法 (LANCZOS)
- ✓ 支持批量处理多个文件
- ✓ 数据完全本地处理，不上传到服务器

## 技术栈

- Pyodide - 在浏览器中运行 Python
- Pillow - Python 图像处理库
